Palo Alto DoS Vulnerability

15th January 2026

Palo Alto have announce a CVSS 7.7 vulnerability ( tracked as CVE-2026-0227 ) in the Global Protect PAN-OS software that permits a denial of service attack due to an improper check for exceptional conditions.

The issue affects:

  • PAN-OS 12.1 < 12.1.3-h3, < 12.1.4
  • PAN-OS 11.2 < 11.2.4-h15, < 11.2.7-h8, < 11.2.10-h2
  • PAN-OS 11.1 < 11.1.4-h27, < 11.1.6-h23, < 11.1.10-h9, < 11.1.13
  • PAN-OS 10.2 < 10.2.7-h32, < 10.2.10-h30, < 10.2.13-h18, < 10.2.16-h6, < 10.2.18-h1
  • PAN-OS 10.1 < 10.1.14-h20
  • Prisma Access 11.2 < 11.2.7-h8
  • Prisma Access 10.2 < 10.2.10-h29

Palo Alto confirmed the issue is applicable only to PAN-OS NGFW or Prisma Access configurations with an enabled GlobalProtect gateway or portal. The exceptional condition allows an unauthenticated attacker to cause a DoS, with repeated attempts resulting in the firewall entering maintenance mode.

Whilst presently the attack does not appear to be actively used in the wild (as yet), it's important to ensure systems are patched as soon as possible.

Chrome Zero Day

18th February 2026

A critical flaw in Chrome has been seen under active attack as Google urges customers to urgently update. CVE-2026-2441 has a CVSS score of 8.8 and has been described as a "use-after-free bug" in CSS, allowing the remote attacker to execute arbitrary code inside a sandbox via a crafted HTNML page.

This is the first actively-exploited zero day in Chrome for 2026 that has been patched, after Google addressed eight last year.

Users are advised to update their Chrome browser to 145.0.7632.75/76 on Windows and Apple MacOS, and 144.0.7559.75 for Linux systems.

Given the reliance on Web-bawed interfaces, the importance on the security and integrity of the browser platform is paramount.

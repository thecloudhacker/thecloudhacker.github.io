Chrome High Severity Patch

21st September 2025

Chrome has had a high severity patch pushed by Google which is under active exploitation. If you haven't already, now is the time to run updates in your browser.

The flaw in the V8 JavaScript and WebAssembly engine, tracked as CVE-2025-10585, is a type confusion bug where the engine misinterprets a block of memory as one type of object when it's actually something else. This flaw can lead to system crashes, arbitrary code execution, and when chained with other outstanding issues, potential full system compromise via a malicious HTML page.

The flaw was discovered and reported by the Google Threat Analysis Group and a patch for Windows, Apple and Linux environments is available now.

JSON Formatter and CodeBeautify Data Leaks

27th November 2025

You would have thought in the current climate of security consciousness that users would have enough savvy to not just blindly paste information into websites online that reformat and provide that information back on an open-access link, but you'd be wrong.

In the continuing case of "finding new ways to exfiltrate data by oblivious stupidity", years of corporate data has been found to have leaked (and remains) on the online tool sites JSONformatter and CodeBeautify. Cybersecurity company watchTowr Labs captured a dataset of over 80,000 files which contained: usernames and passwords; repository authentication keys; Active Directory, Database and FTP credentials; Cloud keys; LDAP configuration inforamtion; helpdesk API keys; meeting room API keys; SSH session recordings and all manner of personal information.

So, just a few items of interest, then.

One might consider that the data might be limited to developers running some early-doors testing, however the companies affected span critical national infrastructure, government, finance, insurance, banking, technology, retail, aerospace, telecommunications, healthcare, education, travel and cybersecurity sectors.

So, let's just say, the world.

In modern news there is so much emphasis placed on elite hackers breaking through tiny cracks in firewalls or security systems whilst a simple script run across online tooling can provide more data readily availale. This isn't so much a back door as a thousand acre farm with an "Open All Hours" neon sign at the open entrance.

Excuse me, I'm off to rotate all my corporate and personal data in case someone thought it was useful to paste bin my life...

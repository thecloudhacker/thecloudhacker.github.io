MS BASIC Code Released

5th September 2025

Microsoft have released the sourcecode for the 6502 BASIC it created way back in 1976 which was utilised by many of the early Microcomputer systems we loved and grew up using such as the Commodore PET, Vic-20 and C64.

Written in Assembly language, the code includes fixes to garbage colletion the Commodore identified. Time stamps on the Github repository show commit dates from 48 years ago, with comedic comments on the repository such as "I see the last commit other than the license was 48 years ago. Is this project abandoned?"

If you're playing around with 6502-based 8-bit systems, you now have a range of Assembly to tinker around with to your hearts content and fork the project.

Discover some retro nostalgia over on Github.

React2Shell Attacks Accelerate

12th December 2025

The React2Shell server-side vulnerability ( CVE-2025-55182 ) that dropped this month is quickly becoming a growing issue for sites across the Internet. With the React Server Components and dependent frameworks ( such as Next.js ) being so utilised across websites, this vulnerability is a huge concern. The unsafe deserialisation allows unauthenticated attackers to send crafted requests to achieve remote code execution. A single exposed endpoint can provide an entry point to larger estates, particularly worrying for multi-hosting servers.

Huntress have a great breakdown of the issue with latest insights into the observed groups exploiting the vulnerability in the wild.

It would appear that a lot of sites are still wide open and vulnerable to the exploit, which (if you haven't patched yet) you should consider pwn3d.

Paolo Alto Networks' Unit 42 have also linked the exploitation to North Korean and Chinese threat groups.

Fun times for all right before the festive period. Patch, patch and patch before Santa Claus and Patch turn up this Christmas!
